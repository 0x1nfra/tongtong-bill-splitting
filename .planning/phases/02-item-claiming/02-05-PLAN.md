  <name>Task 1: Fix @theme inline — emit :root CSS custom properties</name>
  <files>src/app/globals.css</files>
  <action>
    On line 5 of src/app/globals.css, change `@theme inline {` to `@theme {`.

    Root cause: `@theme inline` is a Tailwind v4 directive that makes tokens available as Tailwind utilities but does NOT emit them as CSS custom properties on `:root`. This means `var(--color-stamp)` and `var(--color-pen)` are undefined at runtime, so `text-[--color-stamp]` and `text-[--color-pen]` render with no color.

    `@theme {` (without `inline`) emits all tokens as `:root { --color-stamp: #B91C1C; --color-pen: #1E40AF; ... }` which is what the runtime CSS needs.

    This single character-level change fixes both UAT gaps: ❋ prefix not red (test 1) and claimant names not blue (test 7).

    Do NOT change any token values, variable names, or any other part of the file.
  </action>

  </verify>
